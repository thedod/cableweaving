Cable Weaving

Stories woven out of cables

Download .zip Download .tar.gz Fork on GitHub

CableWeaver lets you see CableGate references between cables in context, and select cables that relate to the same story - thus creating story lines. It's pretty easy to work with (once you read the instructions) and you can easily install it on your own server (or home computer), but in order to demonstrate what it's good for, here are a few story lines with summaries and cable excerpts:

  • Fil mish-mish - the story of Palestine's cellular spectrum
  • Cashless in Gaza - work in progress...
  • Galileo takeover - work in progress...

Journos/bloggers/newsgeeks:

Please note that the content here is in the public domain (see legaleze below). Feel free to publish any page (or part of it) you see here. Credit (to me and/or CableWeaver) could be nice, I'd also love to get a link to whatever you do with this material, but don't feel obliged. Many people have already paid dearly (and are still paying) for releasing these cables. It would be a shame not to extract any insights we can out of them in order to inform the people.

You might also decide to use CableWeaver yourself. You can can then click the export button and generate a story-line pages similar to the ones you see here.
If you create something interesting (or if you have information to add to an existing story), there are 2 ways to contribute it here:

  • You can fork this site and send a pull-request to the gh-pages branch.
  • If you don't understand the previous sentence (it's OK), simply contact me and we'll work something out :)
