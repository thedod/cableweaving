Cable Weaving

Stories woven out of cables

Download .zip Download .tar.gz Fork on GitHub
And Sharkey says: "Hey sport. You connect the dots. You pick up the pieces."

CableWeaver lets you see CableGate references between cables in context, and select cables that relate to the same story - thus creating story lines. These story lines can be exported as a skeleton web page you can edit.
All pages on this site were initially exported from CableWeaver. Quotes, remarks, etc. were then added by me and others (you can help. See below).

Stories so far:

Journos/bloggers/newsgeeks:

Feel free to publish any page (or part of it) you see here, as long as you give credit to the authors (that's what the license below says). If the license is too restrictive for your purposes, just contact me. I [and any co-authors involed] would most probably give you the explicit permission you need, beacuse many people have already paid dearly (and are still paying) for releasing these cables, and it would be a shame for the human race not to extract any insights we can out of.
You may also decide to use CableWeaver yourself. Once you click on the export button, CableWeaver would generate HTML code for a skeleton page similar to this one. You can edit it with any text or html editor, add remarks, quote from the cables, etc.
Tip: as you can see — selected cables on exported graphs are numbered. This done is in order to make story pages print-friendly. It's sometimes handy to print what you have and scribble remarks with a pencil, give someone a copy, etc.

Make this site better

If you want to contribute a new story (or if you have information to add to an existing one), there are 2 ways to do it:
  • You can fork this site and send a pull-request to the gh-pages branch.
  • If you don't know how to do that, would rather stay anonymous, etc. you can contact me securely instead.
